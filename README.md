学习笔记：
概念：

VPN————则是一种技术协议
VPS————Virtual Private Server（虚拟专用服务器），远程的、虚拟化的计算机
机场————提供节点订阅和管理服务的商业销售形式
节点————服务器上运行的一个具体的代理服务实例

计算机————通常指我们日常使用的通用个人电脑
服务器————一种功能特化的计算机（电脑）
区别：
计算机(电脑) ，像一辆私家车，为个人设计，随时开，随时停，主要服务于一个人。
服务器，像重型货车，为特定任务设计。注重载重能力、持续运行时间，服务大量请求。
这两者本质上都是一种“**算力载体**”，所有电子设备，都是这种相同的本质。
